# OmniCore Video ve GIF Senaryoları (Visual Scripts) 🎬

Bu döküman, Yardım Merkezi'ne eklenecek olan görsel rehberlerin (GIF/Kısa Video) adım adım akışlarını içerir.

## 1. Senaryo: İlk Mağaza Bağlantısı (Onboarding)
- **Kamera Açısı:** Ekran kaydı (Fullscreen Dashboard).
- **Akış:** 
    1. Mouse ile sol menüden `Ayarlar`'a tıklanır.
    2. Trendyol anahtarı `Aktif` konuma getirilir.
    3. API anahtarları girilir ve "Bağlantıyı Test Et" butonuna basılır.
    4. "Başarılı" toast mesajı görülür.
- **Odak:** Hız ve basitlik.

## 2. Senaryo: AI ile Ürün Açıklaması Oluşturma
- **Kamera Açısı:** Odaklanmış Ürün Detay Paneli (Sheet).
- **Akış:**
    1. Envanter listesinden bir ürüne tıklanır.
    2. Açıklama alanındaki parlayan "AI ile Optimize Et" butonu seçilir.
    3. Yazıların yavaşça (typing effect) oluştuğu gösterilir.
    4. "Kaydet" butonuna basılır.
- **Odak:** Teknolojik güç (Magic touch).

## 3. Senaryo: Akıllı Fiyat Kuralı Tanımlama
- **Kamera Açısı:** Akıllı Fiyatlandırma sayfası.
- **Akış:**
    1. "+ Yeni Kural Ekle" butonuna tıklanır.
    2. "BuyBox Rakibi" tipi seçilir.
    3. Değer olarak "- %1" girilir.
    4. Kural listesindeki "Aktif" anahtarı açılır.
- **Odak:** Karlılık ve otomasyon.

## 4. Senaryo: Sipariş Onay ve Kargo Barkodu
- **Kamera Açısı:** Siparişler sayfası.
- **Akış:**
    1. "Beklemede" sekmesinden bir sipariş seçilir.
    2. "Onayla" butonuna basılır.
    3. "Kargo Barkodu Al" butonuna tıklanarak PDF önizlemesi gösterilir.
- **Odak:** Operasyonel verimlilik.
